# mao_mind 在线刷题平台

# 使用方式
web：https://buausb.github.io/maogai/

# 功能
* 收藏功能：可以展示标记的题目，时间关系，没有完成统一的展示页面
* 历史记录：每次打开自动定位到上次做题进度
* 动态刷新：做到哪，题目就展示到哪，节省资源
* 漂亮简约的UI设计
